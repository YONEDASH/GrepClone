# Rontgen

A(nother) very basic grep clone written in Go.

## Usage

```
Usage: ./rontgen [-v] [-verbose] <path> <pattern>
Flags:
  -v         Show version
  -verbose   Verbose
  <path>     Path to directory or file
  <pattern>  Pattern to search for
```

## Building

Clone the repository and run ```go build```.
